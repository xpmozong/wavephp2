wavephp2.0 PHP框架 2.0版本

composer.json 配置

    {
        "require": {
            "xpmozong/wavephp2": "*"
        },
        "repositories": [
            {
                "type": "vcs",
                "url": "https://github.com/xpmozong/wavephp2"
            }
        ]
    }

也可以使用composer的安装包列表

    {
        "require": {
            "xpmozong/wavephp2": "*"
        },
        "repositories": {
            "packagist": {
                "type": "composer",
                "url": "http://packagist.phpcomposer.com"
            }
        }
    }
